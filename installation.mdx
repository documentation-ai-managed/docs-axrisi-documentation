## Browser Compatibility

Axrisi is compatible with major browsers including Chrome, Firefox, Edge, and Safari. Ensure your browser is updated to the latest version for optimal performance.

<Callout kind="info">For the best experience, use Chrome or Firefox with the latest updates.</Callout>

## Installation Process

<Steps>
  <Step title="Visit Store" icon="globe">
    Go to the Chrome Web Store or Firefox Add-ons and search for "Axrisi".
  </Step>
  <Step title="Add Extension" icon="plus">
    Click "Add to Chrome" or "Add to Firefox" and confirm the installation.
  </Step>
  <Step title="Grant Permissions" icon="shield">
    Allow the extension access to read and modify webpage content.
  </Step>
  <Step title="Pin to Toolbar" icon="pin">
    Pin the Axrisi icon to your browser toolbar for easy access.
  </Step>
</Steps>

## Troubleshooting Common Issues

<Expandable title="Extension Not Loading" default-open="false">
  Try refreshing the page or restarting your browser. If issues persist, check your internet connection.
</Expandable>

<Expandable title="Permission Denied" default-open="false">
  Ensure you've granted the necessary permissions during installation. You can adjust permissions in your browser settings.
</Expandable>

## Configuration Options

After installation, access the extension settings to customize your experience.

<CodeGroup tabs="Chrome,Firefox">
  ```javascript
// Chrome: Click the extension icon and select settings
chrome.settingsPrivate.set({
  'extension': 'axrisi',
  'autoTransform': true
});
  ```
  ```javascript
// Firefox: Right-click the toolbar and select customize
// Drag Axrisi to your preferred position
browser.settings.open();
  ```
</CodeGroup>
