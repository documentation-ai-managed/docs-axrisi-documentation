## Getting Started with Axrisi

Axrisi is an AI-powered browser extension that transforms text on webpages with a single click. It offers powerful features like summarization, bulletization, explanation, continuation, and correction to simplify complex content and improve productivity. Available in both free and PRO plans, Axrisi helps users enhance their reading and writing experience effortlessly.

## Key Benefits

Axrisi enhances productivity by allowing users to quickly process web content without leaving their browser. The extension integrates seamlessly with popular browsers and provides instant transformations that save time and improve comprehension.

<Callout kind="tip">Get started by installing the extension from the Chrome Web Store or Firefox Add-ons.</Callout>

## How It Works

Select any text on a webpage, right-click to access the Axrisi menu, and choose your desired transformation. The AI processes the content instantly and replaces the original text with the improved version.

## Code Example

Here's a simple example of how the extension processes text:

```javascript
const originalText = "The quantum entanglement phenomenon occurs when particles become linked such that the state of one instantly influences the state of the other, regardless of distance.";
const summarizedText = "Quantum entanglement links particles so their states affect each other instantly, no matter the distance.";
console.log(summarizedText);
```
